# 알고리즘 3팀 길찾기 프로젝트

**C언어 기반 최단경로 탐색 시스템**

C언어와 Dijkstra 알고리즘을 활용하여 구현한 최단경로 탐색 프로그램입니다. 사용자가 출발지와 목적지를 설정하고 교통수단을 선택하면 최적 경로와 예상 소요시간을 제공하는 콘솔 기반 네비게이션 시스템입니다.

### 개발 환경
- **Language**: C Language
- **IDE**: Visual Studio 2019/2022
- **Platform**: Windows 10/11
- **Compiler**: MSVC (Microsoft Visual C++)

### 알고리즘
- **핵심 알고리즘**: Dijkstra's Algorithm
- **데이터 구조**: 그래프, 우선순위 큐
- **파일 처리**: 텍스트 파일 기반 지역 데이터 관리

## 시스템 아키텍처

```
사용자 입력 → 메뉴 시스템 → Dijkstra 알고리즘 → 결과 출력
                   ↓              ↓
            [지역 데이터]    [경로 계산]
                   ↓              ↓
         [Recent Paths.txt] → [결과 저장]
```

다중 메뉴 시스템으로 설계되어 사용자의 다양한 요구사항을 체계적으로 처리합니다. 핵심 Dijkstra 알고리즘을 통해 최단경로를 계산하고, 파일 시스템을 통해 최근 검색 기록을 관리합니다.

## 핵심 기능

### 경로 탐색 시스템
- **출발지/목적지 설정**: 등록된 지역 목록에서 선택
  - 지역명과 별칭을 통한 직관적 선택
  - 잘못된 입력에 대한 재입력 요청
  - 동일 지점 선택 방지 기능
- **교통수단 선택**: 다양한 이동 수단별 최적 경로
  - 도보, 자전거, 자동차, 대중교통 지원
  - 교통수단별 속도 계수 적용
  - 실시간 소요시간 계산
- **최단경로 계산**: Dijkstra 알고리즘 기반 경로 탐색
  - 가중치 그래프를 통한 정확한 거리 계산
  - 중간 경유지 표시
  - 총 거리 및 예상 소요시간 제공

### 데이터 관리
- **지역 정보 관리**: 파일 기반 지역 데이터베이스
  - 지역코드와 별칭 매핑
  - 확장 가능한 지역 추가 구조
- **검색 기록 저장**: 최근 경로 검색 이력 관리
  - Recent Paths.txt를 통한 이력 관리
  - 사용자별 개인화된 검색 기록

### 사용자 인터페이스
- **다중 메뉴 시스템**: 체계적인 기능 분류
  - Menu 1: 경로 검색 및 결과 확인
  - Menu 2: 지역 등록 및 관리
  - Menu 3: 검색 기록 삭제 및 관리
- **직관적 조작**: 숫자 기반 메뉴 선택
- **실시간 피드백**: 즉시 결과 표시

## 데이터 설계

### 지역 구조체 (Region)
```c
typedef struct _Region_ {
    char region[5];      // 지역 코드 (최대 4자)
    char nickname[12];   // 지역 별칭 (최대 11자)
} Region;
```

### 그래프 데이터 구조
- **인접 행렬**: 2차원 배열을 통한 거리 정보 저장
- **정점 정보**: 각 지역의 인덱스와 정보 매핑
- **가중치**: 실제 거리 또는 소요시간 기반 가중치

### 파일 데이터 형식

대구#DGU

## 알고리즘 상세

### Dijkstra 알고리즘 구현

```c
// 핵심 알고리즘 의사코드
1. 시작 정점의 거리를 0으로 설정
2. 모든 정점을 미방문 상태로 초기화
3. 현재 정점에서 인접한 정점들의 거리 계산
4. 가장 가까운 미방문 정점을 다음 정점으로 선택
5. 목적지에 도달할 때까지 3-4 반복
6. 최단 경로 및 거리 반환
```

### 시간 복잡도
- **시간 복잡도**: O((V + E) log V)
- **공간 복잡도**: O(V²)
- **V**: 정점 수 (지역 수)
- **E**: 간선 수 (연결된 경로 수)
